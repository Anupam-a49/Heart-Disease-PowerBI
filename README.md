# ❤️ Heart Disease Power BI Dashboard

> Interactive Power BI dashboard analyzing heart disease clinical data to uncover survival trends, age-group insights, and healthcare risk factors.

---

## 📌 Project Overview

This project presents a visually interactive **Power BI dashboard** built using heart disease clinical data.
The dashboard helps analyze patient survival outcomes, demographic trends, and health indicators through clear KPI cards and dynamic charts.

It is designed to support **data-driven healthcare decisions** and improve understanding of patient risk patterns.

---

## 📊 Dashboard Highlights

✔️ Survival Rate Analysis
✔️ Total Alive vs Total Death Cases
✔️ Age Group Comparison
✔️ Gender-Based Filtering
✔️ Cancer & Diabetes Trend Analysis
✔️ Interactive Visual Insights

---

## 🛠️ Tools & Technologies Used

* Power BI
* Power Query
* DAX Measures
* Data Modeling
* Data Visualization

---

## 📈 Key KPIs

| Metric        | Description                      |
| ------------- | -------------------------------- |
| Survival Rate | Percentage of patients alive     |
| Total Alive   | Number of surviving patients     |
| Total Death   | Number of death cases            |
| Avg Metrics   | Calculated averages for analysis |
